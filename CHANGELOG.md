# CHANGELOG

`CHANGELOG.md` is the source of truth for the current version of `ESSENCE.md`.
This project follows [Semantic Versioning](https://semver.org/): MAJOR for a
change that redefines or removes a law, MINOR for a new law or section, PATCH
for wording/clarity fixes that change no invariant.
